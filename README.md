# Alla Landing Page

A Next.js + Tailwind landing page for Alla Shapiro, focused on fractional R&D leadership, game studio advisory, and team scaling.

## Commands

- `npm install` to install dependencies
- `npm run dev` to start the development server
- `npm run build` to create a production build
